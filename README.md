这里是课程9(AI)作业
-------

实现了AI 

1.在没有玩家出现在视野中时，自动前往一个特别目标地，当玩家出现在视野中时，AI会靠近玩家

2.拥有远程AI(装备枪)和近战AI(装备近战武器)，小Boss(装备火焰发射武器)，当玩家攻击远程AI时，远程AI会躲避，当攻击近战AI/Boss时，近战AI/Boss会加快速度跑向玩家
